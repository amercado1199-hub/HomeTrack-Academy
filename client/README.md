# HomeTrack Academy

HomeTrack Academy is a homeschool management platform built with React, Flask, and SQLAlchemy.

## Features

* User Authentication (Signup, Login, Logout)
* Student Management
* Subject Management
* Lesson Tracking
* Attendance Tracking
* Progress Reports
* User-Specific Data Ownership
* Dashboard Analytics

## Technologies Used

Frontend:

* React
* React Router

Backend:

* Flask
* SQLAlchemy
* Flask Sessions
* Flask Bcrypt

Database:

* SQLite

## Installation

Backend:

pipenv install

python app.py

Frontend:

npm install

npm run dev
